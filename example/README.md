# Examples

This directory contains runnable entry points plus a small number of checked-in reference artifacts.

Use the scripts listed below as the maintained examples. Some `result/` files were generated by older runs and should be treated as reference outputs, not as authoritative input data.

## Layout

| Directory | Purpose | Main entry points |
|---|---|---|
| `example/orr/surface` | ORR on close-packed metal slabs | `run_metals.py`, `make_volcano_plot.py` |
| `example/orr/cluster` | ORR on metal nanoparticles | `run_cluster.py`, `plot_energy_diagram.py`, `make_cluster.py` |
| `example/orr/modified_surface` | ORR on clean and modifier-covered slabs | `code/orr_Pt111.py`, `code/orr_Pt111_modified_surface.py` |
| `example/orr/vasp` | Sample VASP runner and YAML layout | `run_vasp/run_vasp.py`, `data/vasp.yaml` |
| `example/oer/surface` | OER on close-packed metal slabs | `run_metals.py`, `make_volcano_plot.py` |
| `example/oer/cluster` | OER on metal nanoparticles | `run_cluster.py`, `plot_energy_diagram.py`, `make_cluster.py` |
| `example/oer/modified_surface` | OER on clean and modifier-covered slabs | `code/orr_Pt111.py`, `code/orr_Pt111_modified_surface.py` |
| `example/oer/vasp` | Sample VASP runner and YAML layout | `run_vasp/run_vasp.py`, `data/vasp.yaml` |
| `example/cer` | Coupled OER/CER screening on rutile oxides | `code/oxide_oer_cer_.py` |

## Notes

- `surface` examples use slab-like periodic structures.
- `cluster` examples use finite nanoparticles and index-based adsorption sites.
- `modified_surface` examples compare a clean slab with a slab carrying an extra adsorbate or molecular modifier.
- `vasp` directories only provide templates for external VASP execution.
- ORR surface bulk workflows now default to fixed-cell bulk relaxation. Use `bulk_relax_mode="cell_and_positions"` and `bulk_cell_calculator="uma-s-1p2_omat"` when you explicitly want a cell-relaxed precursor.
