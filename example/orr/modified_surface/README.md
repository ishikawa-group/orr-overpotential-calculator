# ORR modified-surface example

This directory compares a clean Pt(111) slab with a Pt(111) slab carrying an extra molecular modifier.

## Entry points

- `code/orr_Pt111.py`: clean-surface ORR reference calculation.
- `code/orr_Pt111_modified_surface.py`: ORR calculation on a modifier-covered surface.

## Notes

- The modified example places `CH3CN` on the slab before evaluating ORR intermediates.
- Results are written under `result/ORR/`.
